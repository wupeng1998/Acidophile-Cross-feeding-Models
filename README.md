# Acidophile Cross-feeding Models

Genome-scale metabolic reconstructions (GEMs) and SMETANA simulations for  
**“Synergistic Iron-Sulfur Metabolism in Two Extremophilic Acidophiles Is Supported by Cysteine Cross-feeding”**

## Files

| File | Description |
|------|-------------|
| `AME_CTT1204.xml` | AC model (two-species community) |
| `AME_CT1761.xml` | ST single-species model |
| `AS_SMETANA.pdf` | SMETANA visualization (vector) |
| `AS_SMETANA.tif` | SMETANA visualization (high-res raster) |
| `detailed.tsv` | Raw SMETANA output (TSV) |
